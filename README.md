# SIGNATE Student Cup 2023
SIGNATE Student Cup 2023【モデリング部門】：中古車の価格予測チャレンジ

## :hugs: Requirement
- Python 3.10.12
- LightGBM 4.0.0
- Matplotlib 3.7.1
- NumPy 1.23.5
- Pandas 1.5.3
- Seaborn 0.12.2
- scikit-learn 1.2.2
